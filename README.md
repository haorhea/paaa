<p>花京院典明十四岁的时候，基本已经形成了一个较为稳定的社交圈，同学，老师，承太郎，承太郎的朋友，承太郎的家人。有时候他自我审视的时候也会觉得有那么几分奇怪，他的社交圈中，并没有“朋友”这个分类。他性格并不孤僻，就算升上国中总算打了耳洞戴上了那副品味堪忧的耳坠，进入了新的环境，他在新学校里依旧是很受欢迎的一个人，然而那些与他在下课时闲扯，放学时同行的同龄人们，花京院更倾向于将他们归入“同学”这个集合中，他真正在心中认定的所谓“朋友”，则包含于“承太郎的朋友”和“承太郎的家人”这两个范围。</p> 
<p>至于突兀地出现在其中的，只有一个值的集合，“承太郎”，正如同幼时便没能搞清楚他对自己来说究竟是父亲，兄长，亦或是叔叔，十四岁的花京院也没有搞清楚如今的承太郎对他来说算是亲人，老师，还是一位朋友，他也懒得搞清楚这种事，没有必要分得那么细致吧，承太郎就是承太郎。</p> 
<p>与那位监护人共同生活了八年之久，对方的社交圈自然而然地影响到了花京院，八年间他认识了承太郎的外公外婆，还有承太郎的舅舅东方仗助。八岁时第一次到美国见到乔瑟夫乔斯达的时候，老先生精神矍铄，经常猛然爆发出一阵响亮的大笑声，把丝吉Q女士逗得跟着咯咯笑起来，同时也把花京院吓一跳，承太郎管他叫“老头”，花京院那时候可看不出他有什么地方像是老头。然而年岁渐渐过去，最近两次见到乔斯达先生的时候，那高大健壮的身躯明显地消瘦佝偻起来，神智有些不清，见到花京院的时候总是颠来倒去地说，嗯？花京院怎么变得这么小了呀？</p> 
<p>花京院对这种误会总不太能理解，因为自己明明每次都比上次来的时候长大了一些。</p> 
<p>至于东方仗助，他们年纪相差不大，初识时他是个高中生，如今做了警察，由于要戴帽子，那不容人批评的发型也只好变了。花京院和仗助虽不常常见面，但见面聊天的时候总觉得投缘，从游戏说到学校，几乎无话不聊，而且多数观点都惊人地相似，两人唯一无法达成共识的大概也只有对空条承太郎其人的印象和看法这一点，东方仗助说你怎么受得了和那个承太郎先生成天住在一起！而花京院答道，你要是天天和他住在一起，就不会有这种担忧了。</p> 
<p>最后说到朋友，花京院不得不提到的还有一个法国人，银发，夸张的耳饰，油腔滑调，香喷喷，若不是承太郎说“我们家不信教”，差点就当了花京院的教父。花京院上国中时，JP波鲁那雷夫已近不惑之年，然而每次他带着一拖轮箱的礼物，活蹦乱跳地坐飞机来日本看花京院的时候，总是表现地比三十出头的承太郎要年轻得多。花京院觉得如果要诚实回答“你最好的朋友是谁”的话，他的答案多半就是波鲁那雷夫了，他与这位忘年交相处的时候时常表现得有些没大没小，而难得的是，这种没大没小并不会引起任何人的不悦，包括他自己，仿佛在见面的一瞬间，他就发现了这就是与这位法国朋友的最佳相处方式一般。</p> 
<p>花京院典明在学校中也有不少聊得来的人，这些人却始终被屏蔽在一个合理的范围之外，虽然关系看上去不错，但接触到的终究并不是最真实的花京院，而要说这层屏蔽的筛选标准是什么，说到底，还是替身。</p> 
<p>替身是精神的力量，因此看不到法皇的人也只能流于表面地与他交往。</p> 
<p>当时国中一年级的花京院还没有这样的认识。</p> 
<p>&nbsp;</p> 
<p>国一升国二的春假结束后，花京院被同班的女生告白了。</p> 
<p>是个性格很活泼的女生，因此过去一年里，和花京院说过的话也不少，外人评价他们，多半能说一句“关系不错”，然而在花京院自己心中，只不过算是“认识”罢了。放学之后他被那姑娘约到了无人的视听教室，女生具体说了什么，他记不大清，开头是“你总算来了，我等了好久”，结尾是“请当我的男朋友”，他一句一句听下来，心跳渐渐快起来，脸也发烫，等对方说完，觉得自己耳朵根都红了。他低着头十分窘迫地说，请，请让我考虑一阵，那姑娘的态度反而要坦荡得多，笑眯眯地说，好的，我等你的答复。</p> 
<p>他骑车回到家，用钥匙开了门之后就一路嚷嚷着承太郎的名字跑进去，房子里一片寂静，那“承太郎承太郎我跟你说”的叫嚷在屋里带着回响，无人应答，花京院这才想起这两天承太郎研究室里都要加班，饭菜虽然都在冰箱里准备好了，但是人并不在家里。</p> 
<p>这个大消息于是在心里憋了一个小时，六点半的时候，花京院看看差不多是晚餐的时间了，抓起家里的座机给研究室打了电话，几经转接，总算通到了承太郎那里。对方第一句话是“我已经忙完了，现在准备出来”，而花京院的回答是“我要有女朋友了”。</p> 
<p>电话听筒那头沉默了一下，花京院于是一股脑把今天发生的所有事都说了出来，承太郎一直静静地听着，没有打断，直到花京院说完之后又过了几秒，他才应了一句：</p> 
<p>“这么说，还没答应？”</p> 
<p>“还……没有……”花京院靠在座机旁边，挠挠后脑，“毕竟我本来也没有喜欢她……”</p> 
<p>“我回来以后你再跟我说详细的吧，晚饭在冰箱里，吃过了没？”</p> 
<p>“正在吃。”</p> 
<p>“吃到一半怎么跑来跟我打电话？”</p> 
<p>“等不及想告诉你嘛，没事，反正我已经吃得差不多了，你快点回来啊！”</p> 
<p>“……知道了。”</p> 
<p>但事实是，承太郎这天回来得很晚，电话挂断三个小时后，等到花京院的作业都做完了，他才按响了玄关的门铃，晚归的理由是临走前突然有了不得不立刻处理的突发事件，因此耽搁了。回家之后，承太郎的话很少，看上去有些疲惫，花京院甚至都在想，他今天好像累坏了，要不明天再跟他说吧，然而等到他放好了包挂好了外套，回到客厅来的时候，承太郎立刻问起了这件事。</p> 
<p>其实在电话里，花京院已经说得够详尽了，承太郎却把问题一个个细化下去：那女生是个怎样的人？认识了很久吗？性格如何？为人如何？花京院在回答的时候才发现他对她几乎全无了解，很多问题都只能含糊地说一句，呃，不错。随着对话的进行，刚开始的兴奋劲渐渐消退下去，等承太郎暂时不再继续提问，他怯生生地回了一句：</p> 
<p>“你是不是不同意？”</p> 
<p>“这种事情无需我同意，谈恋爱是年轻人的自由，关键是你怎么想。”</p> 
<p>“可是你不赞成吧？”</p> 
<p>“我为什么不赞成？你不是说她什么方面都不错吗？”</p> 
<p>承太郎回答的时候，绿眼睛毫不避忌地看着花京院，那眼神似乎是想证明他并没有说谎。这份诚实毫无破绽，令人心焦，因为在花京院的内心深处，他总觉得承太郎应该大力阻拦，强行禁止他和那个女生继续深交才是，接着他便可以因此对承太郎闹一通脾气，最后再顺理成章地回到学校拒绝对方，说是家里不允许，对不起，谢谢，你是个好姑娘。</p> 
<p>花京院十四岁，正是一个典型的，满脑子幻想与逆反的年纪，同龄人多半已经到了反抗期，和家长们话不投机半句多，讲着讲着就会吵起来，花京院暂时没有那个问题，他和承太郎关系很好，说是要对他闹一通脾气，倒是撒娇意味多过迁怒，毕竟平时他也几乎没有机会撒娇，最后也一定会听他的话去拒绝那姑娘。然而承太郎却表现得太开明，太善解人意，明明是放他自行决定，花京院却为此产生了憋着一口气的感觉。他移开视线，盯着自己的拖鞋说，哦，那我可就答应她咯？</p> 
<p>承太郎说，嗯，你自己喜欢就好，慢慢来。</p> 
<p>&nbsp;</p> 
<p>第二天回到学校，他对那位女同学表达了同意，在说的同时，猛然想起来：对了，她并不是个替身使者。</p> 
<p>被告白的兴奋冲昏了头脑，他竟然忘了这点。</p> 
<p>承太郎昨晚怎么也不提醒他一声？</p> 
<p>当然在花京院的人生规划中，并没有明文规定“看不见替身的人不准同我恋爱”，诚然他的确觉得看不见替身的人似乎无法与他推心置腹，然而谈恋爱又是另一码事，电视节目里，书上不也经常出现性格喜好全然不同，但依旧很幸福的情侣或是夫妇吗？巫师难道便不能和麻瓜组建家庭了？</p> 
<p>他既然已经开口答应，女同学又表现得喜出望外，台阶下不来，只好在心里找出千万个理由说服自己：这样也不错。到了下午，消息已经在班级里传开了，最后一节是体育课，课前在更衣室里换衣服的时候，某个关系还不错的同学告诉他说，花京院竟然答应了那女生的表白，不少人可是追悔莫及。</p> 
<p>“追悔莫及？”花京院把体育服往脑袋上套，觉得这词听上去有些别扭。</p> 
<p>“班里还有学校里，对你还有你那继父有意思的人也不少了，只不过你们俩，怎么说呢，有种高岭之花的感觉吧，总觉得就算说出来肯定会被拒绝，所以才都憋着，没想到她一试，竟然这么简单就成功了。”</p> 
<p>男同学用了高岭之花这个词，说完之后自觉幽默，边换衣服边笑起来，满以为花京院会反驳这个带了点揶揄的形容，然而听到的回答，重点却全然不一样。花京院停下了动作，皱起眉头来看着他问道，还有人看上承太郎了？</p> 
<p>“当然，或许还比看上你的人多些呢。”</p> 
<p>“……他才来过学校几次。”</p> 
<p>“那几次可是满城风雨万人空巷。”</p> 
<p>花京院没有回答，从昨天晚上就开始出现的那种，让人很不舒服的感觉越发强烈了。他当然知道承太郎的任何一个方面都非常优秀出众，无论一个女人喜欢男人的脸，还是金钱，还是内涵，空条承太郎都不会让她失望，何况他还非常善解人意，很会照顾别人。</p> 
<p>然而花京院并没有见过这样优秀的承太郎和任何人约会过，他那正当年的监护人在这八年里和恋爱丝毫不沾边，他的生活几乎被工作和陪伴花京院所充满，而花京院也因此很难想象承太郎离开自己，出门去和一个他不认识的女人谈恋爱的样子。或许正是这种难以想象引起了他现在莫名的抵触情绪，不知为何，他非常反感学校里有女生对承太郎产生了兴趣这一事实。</p> 
<p>仿佛光是她们单方面的一厢情愿，就会产生可乘之机，随时要把自己最重要的宝物抢走似的。</p> 
<p>放学的时候，他和女友同路一段，接着在某个路口分开，各自回了家，路上自然也聊了不少，各自的兴趣爱好或是今天上课出糗的老师之类，对话还算愉快，然而等花京院说自己问了承太郎的意见之后，女生似乎十分惊讶。</p> 
<p>“你告诉你爸了！”</p> 
<p>他不是我爸，花京院心想，不过他懒得细究，因此耸耸肩问道，怎么了，不行吗？</p> 
<p>“这种事谁会立刻就告诉家长啊！你也真是个好孩子！”</p> 
<p>客观来说，花京院的确是个好孩子，但是他并不赞同这句话。他与承太郎说这件事，并非像是那些所谓的“好孩子”多年来迫于父母的威慑才不得已坦白招供，而是以平等的，朋友一般的关系交流信息，是他主动想让承太郎知道的，而承太郎也表现得很理解——或许理解得过了头。</p> 
<p>然而他这样解释了之后，姑娘却没有被说服，她一路都难以置信，连连摇头，直到那个路口，两人分道扬镳，而花京院也觉得再说她也不会明白。他回到家之后照例去书房看了会儿闲书，又把作业稍微完成了一些，六点半不到，门口传来钥匙开门声，他探头对玄关处的承太郎招呼一声说，你回来啦。</p> 
<p>我回来了。监护人应了一句，换好鞋走进室内的时候又问他：今天怎么这么早就回来了？</p> 
<p>“早？和平时一样吧？”</p> 
<p>“不是交了女朋友吗？没送她回家吗？”</p> 
<p>“你念高中的时候谈恋爱，都是需要每天放学送对方到家门口的吗？”</p> 
<p>“……不，也没有。”</p> 
<p>承太郎脱下白色的外套，换上深色的居家服进了厨房。花京院在原地站了一会儿之后，跟着他走进去，看着他切菜热锅的背影犹豫许久，问道：这么说，你高中的确是谈过恋爱的？</p> 
<p>他问完之后很紧张，反倒是对面砧板上切菜的声音一点都没停下，咄咄咄咄，很有规律地继续响着。承太郎说，嗯，谈过。</p> 
<p>“和谁？！”</p> 
<p>“你不认识，时间也不长，两个月都不到就分了。”</p> 
<p>“之后呢？就这一次？”</p> 
<p>“就这一次。”，</p> 
<p>“是个怎样的人？”</p> 
<p>“是个替身使者。”承太郎停下了手中的刀，回头看着他，“你那位呢？”</p> 
<p>“……她是个普通人，看不见法皇的。”花京院回答得有些沮丧，仿佛在比赛中棋差一招，输了什么似的，承太郎转回去重新动起了菜刀，边切青椒丝边说，也无需在意，替身使者要是只能和替身使者交往，那我们要绝种了。</p> 
<p>这说法和花京院早上试图说服自己的时候几乎一模一样，可由承太郎说出来，说服力反而弱了一些。国中生凑上前去说，那我就一辈子把替身的事瞒着？不告诉她？承太郎回头看了他一眼问道，才第一次交女朋友，你已经在想一辈子了？</p> 
<p>“打个比方而已。”</p> 
<p>“瞒一辈子，也不大可能，你最好试探性地问一下，如果有这样的东西存在的话，她是否能理解，是否能接受。能不能看见并不是最重要的，关键是心意。”</p> 
<p>监护人的回答仿佛教科书一样冠冕堂皇，让人挑不出刺来，花京院于是便点了点头。当晚吃饭时又问了些恋爱时该怎么做这样的问题，承太郎的回答比较含糊，多数是“看她的眼睛，了解她在想什么”这样的无理要求，花京院最后叹了口气，说她又不是你，我又不是超能力者，我怎么可能看她的眼睛就知道她在想什么。而承太郎依旧还是那句话，他淡淡地说，没关系，慢慢来。</p> 
<p>&nbsp;</p> 
<p>花京院典明的初恋夭折于一个月之后。</p> 
<p>这一个月的时间里，他试着和女友说起替身的事。假如，我是说，假如，如果我的身后有一种超自然的力量，他可以帮我拿到很远处的东西，我失落的时候他会安慰我，总之，我并不是普通人，这样你能接受吗？为求婉转，自然说得含含糊糊的，也有些词不达意。</p> 
<p>女友的回应一开始是，怎么了，是什么笑话吗？测试吗？动画吗？提得多了，女生渐渐不耐烦起来，十分严厉地指责花京院，都已经是国中二年级的人了，不要再沉浸在那种幻想中，意淫自己是个英雄什么的，那是小孩子的玩意，十四岁了还玩这一套，太幼稚了，丢脸。</p> 
<p>花京院最后一次抗辩的时候，她索性爆发，甚至扬言要改变花京院的这种性格，“让他更像个男人”才行，接着又吐露了对他一直以来对承太郎不反抗，什么都告诉他的行为也是忍受已久。</p> 
<p>你要当乖宝宝到什么时候？她咄咄逼人地问他，他说什么你都会听吗？你这是被他洗脑了！</p> 
<p>花京院像被踩了尾巴，很恼怒地说承太郎才没有洗我的脑。他几乎是在吼了，这或许是他第一次在公众场合动怒，何况对象还是个女孩子，对方也一惊，讶异的表情维持了半秒，眉头挑起来，似乎是要回骂，花京院却赶在他开口之前，余怒未消地加了一句：</p> 
<p>“分手吧。”</p> 
<p>说完他扭头就走了。</p> 
<p>被告白，接受，交往一个月后又甩了女生，无论怎么看都像是个薄情男人才会做的过分事，向来对女生相当温柔的花京院典明原本并不会这么干，他觉得自己多多少少也该感到一些愧疚吧，然而离开的路上，一腔酸涩中没有后悔，内疚也很少，更多的是自己的意义被否定时的疼痛。这是法皇觉醒之后，他第一次试图向普通人解释自己的特殊之处，而这次尝试却狠狠碰了壁，正如多年前承太郎告诉他的，他们看不见，不仅看不见，还要因为自己的看不见而要求抹杀这种事物本身的存在。</p> 
<p>他早就过了动不动眼泪汪汪的年纪，但这次碰壁的疼痛超出了预想，下午的课便无心上了，到医务室开了张假条，中午就回了家。到家开门进去的时候闻到一股烟味，他咳嗽一下，鞋子都没换就冲进家门，心想该不会是着火了吧，到了客厅才发现承太郎原来在，那烟味是他在抽烟。</p> 
<p>承太郎会抽烟？</p> 
<p>监护人扭头看到他，还剩大半截没有烧完的烟摁灭在烟灰缸里，他们对视一会儿，同时开口问对方，你怎么回来了？</p> 
<p>承太郎先回答，说是今天研究所下午没事，就放人回来了。答毕轮到了花京院，国中生不知要怎么说，也不愿意谎称自己身体不适，他身体不错，但每次生病，承太郎都不知道有多着急。</p> 
<p>“我……”</p> 
<p>“失恋了吧？”</p> 
<p>“……我甩的她。”</p> 
<p>对方像是有读心术似的一语道破天机，但也没必要问什么“你怎么知道”，承太郎从沙发上站起身走到他面前，一根手指划过他的眼角，笑了笑说，哭什么。花京院抽着鼻子说，她为什么看不见法皇？</p> 
<p>“你很希望她能看见吗？”</p> 
<p>“……无所谓。”他胡乱地在眼角鼻头抹了一把，扑过去抱住了承太郎，“我爱你。”</p> 
<p>承太郎拍拍他的后背，似乎是被逗笑了。</p> 
<p>“因为我看得见法皇？”</p> 
<p>“不单单。因为你养大了我。而且你对我很好。谢谢。”</p> 
<p>“你已经十四岁了，不能再随随便便对人说我爱你了，花京院。”</p> 
<p>“我爱你。”</p> 
<p>“我也爱你……”承太郎在一片烟味中揉了揉他的后脑，叹了一口气答道，“所以不用谢，也别太难过了。”</p>
